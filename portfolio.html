Rescue Spot

Rescue Spot was created to help the user search for dogs that are available for adoption in their desired area with their zip code. Not only does it allow the user to find dogs, but it also allows the user to see local dog parks to enjoy with their newly adopted pet.

Day Planner

Day Planner was created to help our user to schedule and add their events for their work day. The events are saved on local storage which allows the user to retain the items even if the page has been refreshed.
Deployed Link | Github Repository

Coding Challenge

Coding Challenge is a timed quiz that asks the user a couple of coding questions. The user clicks the start button to begin the quiz and has until the end of the timer to complete the questions.
